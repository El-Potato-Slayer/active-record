# Active Records Model of Micro-Reddit

This project is an application similar to Reddit (called active_record) where a user can create a post and add comments to it.



## Build-With

- Ruby -v  2.6.6
- Rails -v 6.0.3.4
- SQLite3
- VScode


## Getting Started

### Prerequisites

To get this project up and running locally, you must already have ruby and necessary gems installed on your computer

**To get this project set up on your local machine, follow these simple steps:**

1. Open Terminal.
2. Navigate to your desired location to download the contents of this repository.
3. Run ```git clone https://github.com/El-Potato-Slayer/active-record.git```.
4. Run ```cd active_record```.
5. Run ```bundle install``` to get the necesary gems.
6. Run ```yarn install```.
6. Run `rails db:migrate`.
